# 🌸 Stayover Countdown

お泊まり会に向けた  
カウントダウン＆しおりWebページ。

楽しみは、当日より前から始まっている。

---

## 🎯 このプロジェクトの目的

- お泊まり会までのワクワクを可視化する  
- 当日のスケジュールを整理する  
- 持ち物チェックで忘れ物を防ぐ  
- ページ自体も思い出の一部にする  

「ただの静的ページ」ではなく、  
時間と一緒に変化する小さな体験設計を目指しました。

---

## 🛠 使用技術

- HTML
- CSS
- JavaScript（Vanilla）
- localStorage（チェック状態保存）
- requestAnimationFrame（アニメーション）

フレームワークは使用していません。  
構造理解と設計練習のため、素の構成で実装しています。

---

## ✨ 主な機能

### ⏳ カウントダウン
- 指定日時までの残り時間を表示
- 日 / 時間 / 分単位で更新

### 📅 Day1 / Day2 切り替え
- タブでスケジュール表示を切替
- アクティブ状態をCSSクラスで制御

### ✅ 持ち物チェックリスト
- チェック状態をlocalStorageに保存
- リロードしても保持される

### 🌸 アニメーション演出
- requestAnimationFrameで軽量描画
- 過剰演出を避けた粒子設計
